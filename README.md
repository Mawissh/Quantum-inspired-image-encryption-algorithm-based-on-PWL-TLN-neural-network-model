# Quantum-inspired-image-encryption-algorithm-based-on-PWL-TLN-neural-network-model
A quantum image encryption framework is proposed using a PWL-TLN chaotic model and PWLTLN-COGSA-based dynamic S-box construction. The proposed PWL-TLN model is validated through Lyapunov exponents, bifurcation diagrams, phase portraits, NIST tests, and analog circuit implementation. A coupled diffusion-substitution mechanism combining CSGC pre-diffusion, PWL-TLN-driven quantum XOR diffusion, and block-wise S-box substitution enhances plaintext sensitivity and nonlinear confusion. Experimental results show near-ideal entropy, near-zero adjacent-pixel correlation, and strong resistance against differential and chosen-plaintext attacks.
